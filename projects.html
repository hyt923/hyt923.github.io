慢肾小谣言

  1. 主页
  2. 慢肾小谣言

1.患有肾病就要补肾：患有肾病并不等于需补肾，盲目补肾反而会伤害肾脏。是药三分毒，几乎所有的药物都会在肾脏中排出，若滥用保健品或补肾药物都会增加肾脏负担。

Read More

2.肾病一定会发展到尿毒症阶段：确实慢性肾病可发展到尿毒症阶段，但这并不是绝对的。相当一部分肾病病人只要及时纠正坏习惯，控制好饮食，规范性用药，定期去医院做检查就能稳定病情。

Read More

3.荤汤营养价值高：很多人喜欢喝排骨汤或各种海鲜汤等，认为营养价值高，能补肾。然而汤煲的时间越长，营养价值就越低，甚至有有害物质析出。长时间煲汤食物中的嘌呤、脂肪以及无机盐会进入汤中，无疑会增加肾脏负担，尤其是伴有身体水肿、三高、少尿或无尿以及尿酸患者更不能喝。

Read More

4.肾病患者不能吃豆制品豆制品：如豆腐或豆浆等，其中属于优质蛋白质，适合肾友。大豆中蛋白氨基酸模式跟人体氨基酸模式差不多，含有9种人体所需的氨基酸。虽然大豆中的蛋白质属于植物蛋白，不过吸收率和利用率较高。另外大豆中含有的大豆异黄酮具有抗氧化和降血脂效果，因此肾病患者可以适量吃。

Read More

5.肾病病人不能吃降压药：很多肾病患者伴有高血压，持续性高血压会对心脏和大脑带来不可逆的损伤，因此需合理用降压药物来稳定血压，常用的是沙坦类和普利类药，不仅能降血压，而且也能保护肾脏，防止肾功能恶化。

Read More

6.伴有高血压和水肿的患者需吃低钠盐：肾病病人不能吃低钠盐，因为低钠盐中含有太多钾，易引起高钾血症，特别是慢性肾功能不全和肾衰竭病人，可能会因为高钾血症而出现心脏骤停，最终威胁生命。

Read More

7.肾病患者不能吃海鲜：其实只要肾病患者对海鲜不过敏，没有痛风发作或高尿酸血症可以少量吃海鲜，能为身体补充所需优质蛋白质。不过痛风、伴有高尿酸血症，还有慢性肾功能患者尽量少吃为妙。

Read More

8.早晨喝一杯淡盐水有利于肾病患者健康：对于正常人来说晨起喝杯淡盐水能促进胃肠道蠕动，健康又长寿。然而肾病患者喝淡盐水可增加肾脏和血管负担，甚至会加重水肿和高血压，早晨喝杯清水即可。

Read More

9.某种方法能彻底治愈肾病：目前为止没有任何特效药物或疗法能彻底治愈肾病，尤其是已经硬化的肾小球是不可逆的过程。
